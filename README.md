# GitGud
An LLM-based github helper, GitGud. Built as the final project for the PESU I/O Gen AI course.


IdeaMap:
- Repo-Ready Code + Workflow Generation [File Structure + env + etc]
- Analyzes pre-existing repos and gives suggestions
- Git Command Assistance / Execution
- Attachments:
    - Repos
    - Files
    - Websites (webscraping)	-> v2
    - Multimedia	-> v2
- Merge-Conflict Resolver
- CI/CD Integration -> v2
